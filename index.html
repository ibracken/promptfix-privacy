Privacy Policy

Last updated: February 26, 2026

PromptFix is a browser extension that helps users rewrite prompts in ChatGPT and Claude.

What data PromptFix handles

Website content: PromptFix reads and replaces text in the active chat input field on supported sites (chatgpt.com, chat.openai.com, and claude.ai) only when the user clicks extension actions.

Account/payment status: If premium features are enabled, PromptFix uses ExtensionPay to check subscription status. ExtensionPay may process account and payment-related data under its own terms.

Extension storage: PromptFix uses Chrome extension storage for minimal extension state required for functionality (for example, premium/license status checks).

How data is used

Data is used only to provide the extension's single purpose: upgrading prompt text in supported chat interfaces and managing premium feature access.

Data sharing

We do not sell user data.

We do not transfer user data to third parties except as required to provide core functionality (for example, payment/subscription verification through ExtensionPay).

We do not use or transfer user data for advertising, creditworthiness, or lending decisions.
